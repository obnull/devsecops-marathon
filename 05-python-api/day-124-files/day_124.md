## Key Concepts
- `with open()` — context manager, closes file automatically
- `json.load()` / `json.dump()` — parse and serialize JSON
- `csv.DictReader` — reads CSV rows as dictionaries
- Open mode `"a"` — append without overwriting existing content

## Takeaways
1. Always use `with open()` — never manual `f.close()`
2. `csv.DictReader` is cleaner than manual split for structured data
3. Append mode "a" is standard for log files
4. `os.makedirs(exist_ok=True)` prevents crash if directory already exists
