## Description
Built port scanner with classification and log filter with aggregation. Used for loops, while with retry logic, break/continue, list comprehensions.

## What Was Done
- Wrote port_scanner.py — range iteration, classification, suspicious port detection, retry demo
- Wrote log_filter.py — parsing, filtering, aggregation, per-IP stats
- Used enumerate() for indexed output
- Implemented while loop with break/else for retry pattern
- Applied list comprehensions for filtering
- Used continue to skip static files
- Used break to find first occurrence

## Commands Used
- `python3 src/port_scanner.py`
- `python3 src/log_filter.py`

## Verification
- Port range iteration correct
- Suspicious ports flagged
- Retry logic works, break on success
- 7 error entries found
- Static files skipped via continue
- First server error found via break

## Issues Encountered
None.

## Key Concept
Loops + conditions = 80% of DevOps automation. Iterate servers, filter logs, aggregate metrics, find anomalies.

## Takeaways
1. for is primary; while is for retry/wait scenarios
2. enumerate() gives index + value
3. break exits, continue skips iteration
4. List comprehensions cleaner than for-loop append
5. while's else runs only if break didn't fire
