# Stage 01: Linux & Git Basics
## Day 09: File Permissions (Deep Dive)

**Description**

Deep dive into Linux file permissions. Practoced numeric and symbolic chmod, understood umask, explored SUID/SGID and dangerous permissions.

**What was done:**

- Set permissions using numeric (640) and symbolic (u=rw,g=r,0=) modes
- Analyzed umask and why new files get 644, directories get 755
- Tested insecure 666 and discused risks
- Found SUID binaries on system

### Result

Understood how permissions work under the hood and why they matter for security.
